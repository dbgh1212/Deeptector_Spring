# Deeptector_Spring
#####  DeepTector는 실시간 행동 분류기 입니다. 이 repository는 Spring Server에 대한 내용입니다.
<br/><br/>
## 구조
- RestFul Api<br/><br/>
  >Web Server와 RestFul기반으로 HTTP 통신을 합니다.
- Socket 통신<br/><br/>
  >DeepLearning Server, Android와 소켓통신을 합니다.
 <br/><br/>
